v1.0 Release Notes
================================

Highlights in 1.0:

* Launch interactive HPC Desktop sessions with new dashboard "batch connect" plugin
* PBSPro support
* LSF 9.1 partial support (We currently don't have access to an LSF 9.1 system to properly test our LSF adapter.)
